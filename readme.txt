--------------------instruction------------------

Prerequisite:
1. install java 1.6-1.8
2. download RMIProj zip file and unzip it

Run the project:

1. Open the RMIProj eclipse project in eclipse

2. Compile the java classes first
	
	a. java *.java
	
	b. rmiregistry

	c. run the server code (make sure the port
	   is not used already)

3. Run the client, follow the guide output of client 
   program, and input the Integer list you want to sort
